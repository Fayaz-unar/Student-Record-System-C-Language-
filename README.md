# Student-Record-System-C-Language-
A robust, console-based Student Management System developed in C. This application allows users to manage student data through a simple menu-driven interface with persistent data storage.

🚀 Features
CRUD Operations: Full support for Adding, Viewing, Updating, and Deleting student records.

Persistent Storage: Automatically saves and loads data from students.dat using binary file handling.

Search Functionality: Quickly find specific students using their unique Roll Number.

Data Integrity: Includes buffer clearing and string handling to ensure clean user input.


🛠️ Technical Details
Language: C

Data Structure: struct for student attributes (Name, Roll Number, Marks).

File Handling: Binary I/O (fwrite/fread) for efficient data storage.
